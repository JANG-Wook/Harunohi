# Harunohi (하루노히)

> 누구나 쉽게 만들고, 엔터프라이즈도 만족할 수 있는 **하이브리드(시나리오 + RAG) 챗봇 빌더 SaaS**

## 한 줄 정의

비개발자도 시각적으로 챗봇을 만들고, 코드 몇 줄로 자사 웹/앱에 임베드할 수 있는 챗봇 빌더 SaaS.

## 핵심 차별화

- **쉬움** — 비개발자/엔터프라이즈 운영자 모두를 위한 직관적 빌더
- **하이브리드** — 시나리오의 결정론적 안정성 + RAG의 자연스러움
- **임베드 우선** — `<script>` 한 줄로 고객사 웹/앱에 챗봇 추가
